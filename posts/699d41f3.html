Hexo Front-Matter简介

发表于2024-10-22|更新于2025-01-16|前端Hexo
|浏览量:

Front-matter是markdown文件上方以---分割的区域，用于指定当前文章参数

  • Page Front-Matter 用于单独页面配置
  • Post Front-Matter 用于文章页配置

Hexo Page Front-Matter选项

可以配置参数如下

配置选项参数 默认值 配置选项参数说明
title 「必须」页面标题
date 「必须」页面创建日期
type 「必须」标签、分类和友情链接三个页面需要配置
updated 「可选」页面更新日期
description 「可选」页面描述
keywords 「可选」页面关键字
comments true 「可选」显示页面评论模块
top_img false 「可选」页面顶部图片
mathjax false 「可选」显示mathjx
katex false 「可选」显示katex
aside true 「可选」显示侧边栏
aplayer 「可选」在需要的页面加载aplayer的js和css
highlight_shrink 「可选」配置代码框是否展开

Front-matter选项中所有内容均为非必填项，建议至少填写title和date值

Hexo Post Front-Matter选项

配置选项参数 默认值 配置选项参数说明
title markdown文件标题 「必须」文章标题，强烈建议填写该选项
date 文件创建日期 「必须」文章创建日期，强烈建议填写此选项，且最好保证全局唯一
updated 文件更新日期 「可选」文章更新日期
tags 「可选」文章标签，一篇文章可多个标签
categories 「可选」文章分类，主题分类显示宏观大分类，建议一篇文章一个分类
keywords 文章关键字 「可选」文章关键字，SEO时需要
description 文章描述 「可选」文章描述
top_img 「可选」文章顶部图，推荐使用图床来做路径
cover false 「可选」文章缩略图，（未设置top_img，文章页顶部将显示缩略图，可设置false，图片地址，留空）
comments true 「可选」显示文章评论模块，默认true
toc toc的enable配置 「可选」显示文章TOC（默认为设置中TOC的enable配置）permalink_defaults:是否开启TOC，可以针对某篇文章单独关闭TOC功能。前提是主题配置文件中激活了toc选项
toc_number toc的number配置 「可选」显示toc_number
toc_style_simple 「可选」显示toc简洁模式
copyright true 「可选」显示文章版权模块（默认为设置中post_copyright的enable设置）
copyright_author 「可选」文章版权模块的作者
copyright_author_href 「可选」文章版权模块的文章作者链接
copyright_url 「可选」文章版权模块的文章链接
copyright_info 「可选」文章版权模块的版权声明文字
mathjax false 「可选」显示mathjx
katex false 「可选」显示katex
aside true 「可选」显示侧边栏
aplayer 「可选」在需要的页面加载aplayer的js和css
highlight_shrink 「可选」配置代码框是否展开

如果top_img属性不填写，文章特色图片会取文章标题的hashcode取余，然后选取主题对应的特色图片，从而达到让所有文章的顶部图各有特色。
date值尽可能的保证唯一，因为某些主题的Gitalk和Gitment识别id是通过date值作为唯一标识符
